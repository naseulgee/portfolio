2. 설치 방법

다운받아진 [SkypeSetupFull]을 실행시켜 줍니다.

설치하기 팝업이 뜨면 언어를 선택할 수 있는 옵션이 나타납니다.
기본 설정인 [한국어]로 둔 상태에서 [동의합니다-다음] 버튼을 클릭합니다.
만일 프로그램의 설치 경로를 변경하시고 싶으신 분들은 [추가 옵션]을 눌러서 변경해주세요.

SKYPE 설치와 관련 없는
[Bing을 내 검색 엔진으로 설정]
[MSN을 홈페이지로 설정하기]
체크박스를 풀어주세요.

프로그램이 잘 설치되고 있습니다!
프로그램 설치가 완료되면 로그인을 진행해주시면 됩니다.

3. 계정 만들기

가입 때 적었던 ID 혹은 E-mail 주소로 로그인 합니다.
계정이 아직 없으신 분들은 아래의 [계정만들기]를 선택해주세요.

SKYPE 가입시에는 반드시 전화번호가 없어도 괜찮습니다.
로그인 하실 때 전화번호가 기억하기 쉬우신 분들은 그대로 진행해주세요.
전화번호가 없거나 이메일로 로그인하시고 싶으신 분들은 [전자 메일을 대신 사용]을 눌러주세요.

입력이 완료되었으면 [동의하고 계정 만들기]를 클릭해주세요.

성과 이름을 입력 후 다음버튼을 눌러주세요.

국가/지역은 한국을 선택 후 생년월일을 입력해주세요.

입력하신 전화번호/이메일로 인증코드가 발송되었습니다.
받으신 코드를 입력한 후 다음단계로 진행합니다.

이제 두 가지 정도의 설정만 하면 사용 가능하십니다!
!스카이프를 사용하기 위해서는 기본적으로 마이크와 웹캠이 필요합니다!
근래 출시되는 대부분의 노트북에는 마이크와 카메라가 내장되어 있으니 큰 문제가 없을 것입니다.
계속버튼을 눌러 진행해주세요.

스피커 테스트를 위해 위의 네모 박스부분을 눌러주세요.
소리가 제대로 들린다면 정상입니다.
오른쪽 카메라의 영상이 제대로 나오는지도 확인해주세요.

마이크 테스트를 위해 마이크에 말을 해보세요.
위의 그림과 같이 음성 인식바가 나타나면 정상입니다.
테스트가 완료되면 계속 버튼을 눌러서 다음 단계로 넘어가주세요.

프로필 사진을 추가하는 단계입니다.
계속 버튼을 눌러 진행해주세요.

웹캠에 자신의 얼굴이 나오도록 한 뒤 [사진찍기] 버튼을 눌러주세요.
만약 다른 이미지나 사진을 원한다면 [찾아보기] 버튼으로 선택해주세요.

사진 선택이 완료되었다면 [이 사진 이용] 버튼을 누릅니다.

이제 모든 설정이 마무리 되었습니다~!
[Skype 사용 시작] 버튼을 눌러 이용해주세요~
